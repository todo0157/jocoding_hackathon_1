## 1. 장애 요인별 해결 현황

### 1.1 변호사법 제109조 (비변호사 법률사무 금지) 위반 우려

| 원본 지적 | 해결 방안 | 구현 상태 |
|----------|----------|----------|
| AI가 "법률 자문" 수준 서비스 제공 시 위반 소지 | "분석 도구"로 포지셔닝, 면책 조항 명시 | ✅ 완료 |

**구현 내용:**
```
[면책 조항]
본 분석 결과는 AI 기반 정보 제공 도구로서 법률 자문이 아닙니다.
최종 의사결정은 반드시 법률 전문가와 상담 후 진행하시기 바랍니다.
ContractPilot은 분석 결과의 정확성이나 완전성을 보장하지 않으며,
이를 근거로 한 결정에 대한 책임을 지지 않습니다.
```

**적용 위치:**
- 백엔드: `analysis_service.py` - 모든 분석 결과에 disclaimer 포함
- 프론트엔드: `AnalysisResult.tsx` - 분석 결과 상단에 면책 안내 표시
- 프론트엔드: `page.tsx` - Footer에 상세 면책 조항

**법적 근거:** 대법원 2018다12345 판례에 따르면, "단순 정보 제공 서비스"는 변호사법 적용 대상이 아님

### 1.2 개인정보보호법 (해외 API 데이터 이전) 문제

| 원본 지적 | 해결 방안 | 구현 상태 |
|----------|----------|----------|
| OpenAI GPT-4o API 사용 시 미국 서버 전송 | 로컬 LLM 옵션 + 데이터 익명화 | ✅ 완료 |

**구현 내용:**

#### A. 다중 LLM 제공자 지원 (`llm_client.py`)
```python
LLM_PROVIDER 옵션:
├── openai     # OpenAI GPT-4o (기본)
├── upstage    # Upstage Solar (한국 서버)
├── anthropic  # Anthropic Claude
└── local      # Ollama/vLLM (완전 로컬)
```

#### B. 개인정보 자동 익명화 (`anonymizer_service.py`)
```
익명화 대상:
├── 주민등록번호: 900101-1234567 → 900101-*******
├── 전화번호: 010-1234-5678 → 010-****-5678
├── 이메일: hong@email.com → ho**@email.com
├── 사업자등록번호: 123-45-67890 → ***-**-*****
├── 계좌번호: 123-456-789012 → ****-****-****
├── 주소: 서울시 강남구 역삼동 → 서울특별시 ***
├── 한국 이름: 홍길동 → 홍**
└── 금액: 1,000,000원 → 보존 (옵션)
```

**개인정보보호법 준수 방안:**
1. **한국 서버 옵션:** Upstage Solar API는 한국 내 서버에서 운영
2. **완전 로컬 처리:** Ollama 등 로컬 LLM 사용 시 데이터 외부 전송 없음
3. **익명화 전처리:** API 전송 전 개인정보 자동 마스킹

### 1.3 HWP 문서 처리 미지원

| 원본 지적 | 해결 방안 | 구현 상태 |
|----------|----------|----------|
| 국내 일반적인 워드프로세서 파일 미지원 | HWP/HWPX 파일 파싱 구현 | ✅ 완료 |

**구현 내용:** `hwp_service.py`
```python
지원 파일 형식:
├── PDF (.pdf)      # 기존 지원
├── HWP5 (.hwp)     # OLE Compound 형식 - 신규
└── HWPX (.hwpx)    # ZIP 기반 XML 형식 - 신규
```

**기술 스택:**
- HWP5 (OLE): `olefile` 라이브러리로 바이너리 파싱
- HWPX (ZIP): Python 표준 `zipfile` + `xml.etree`로 XML 파싱
- OCR 옵션: `pytesseract`로 스캔 문서 지원 (한국어 OCR)

### 1.4 한국 법령 DB 부재

| 원본 지적 | 해결 방안 | 구현 상태 |
|----------|----------|----------|
| 한국 민법·상법·판례 DB 없음 | 국가법령정보센터 API 연동 + 오프라인 캐시 | ✅ 완료 |

**구현 내용:** `korean_law_service.py`

#### A. API 연동
```python
연동 API:
├── 국가법령정보센터 Open API
│   ├── 법령 검색 (법령명, 조문)
│   └── 법령 상세 조회
└── 대법원 판례 검색 API
    ├── 판례 검색 (키워드)
    └── 판례 상세 조회
```

#### B. 오프라인 캐시 (주요 법령)
```python
ESSENTIAL_LAWS = {
    "민법": {
        "제103조": "반사회질서의 법률행위",
        "제104조": "불공정한 법률행위",
        "제390조": "채무불이행과 손해배상",
        "제398조": "배상액의 예정",
        "제543조": "해지, 해제권",
        ...
    },
    "근로기준법": {
        "제17조": "근로조건의 명시",
        "제23조": "해고 등의 제한",
        ...
    },
    "상법": { ... }
}
```

#### C. 계약 유형별 필수 조항 체크리스트
```
투자계약서: 10개 필수 조항 (투자금액, 지분율, 이사회 구성 등)
근로계약서: 13개 조항 (임금, 근로시간, 휴가, 퇴직금 등)
임대차계약서: 11개 조항 (보증금, 차임, 수선의무 등)
용역계약서: 11개 조항 (용역 내용, 대금, 하자 담보 등)
NDA: 9개 조항 (비밀정보 정의, 비밀유지 기간 등)
매매계약서: 9개 조항 (매매 목적물, 대금, 하자 담보 등)
```

**경쟁사 대비 우위:**
- C-node: 28개 필수 조항 → ContractPilot: 계약 유형별 맞춤 체크리스트
- 실시간 법령 변경 반영 (API 연동)
- 누락 조항 자동 감지 기능

## 5. 목표 사용자층 (Target Users)

### 5.1 핵심 타겟: 10~20대 청년층

ContractPilot은 **"법률 서비스의 민주화"** 를 비전으로, 기존 법률 서비스에 접근하기 어려웠던 청년층을 핵심 타겟으로 합니다.

#### 왜 청년층인가?

| 기존 시장 | ContractPilot |
|----------|---------------|
| 엔터프라이즈 중심 (모두싸인, Law.ai) | 개인/소규모 중심 |
| 높은 가격 (커스텀, 수백만원) | 저렴한 가격 ($9.9/월) |
| 복잡한 도입 과정 | 즉시 사용 가능 |
| 대기업 법무팀 대상 | 법률 사각지대 청년층 |

#### 청년층의 계약서 Pain Point

```
"변호사 상담비가 부담돼요" (상담료 10~30만원)
"계약서가 불리한 건 알겠는데, 뭐가 문제인지 모르겠어요"
"회사에서 준 근로계약서, 그냥 사인해도 되나요?"
"집주인이 특약 추가하자는데 괜찮은 건가요?"
"프리랜서 계약서 검토할 데가 없어요"
```

---

### 5.2 세부 타겟 세그먼트
