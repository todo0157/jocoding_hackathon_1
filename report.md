## 서비스 개요

**ContractPilot**은 AI를 활용하여 계약서의 위험 조항을 자동으로 분석하고, 한국 판례를 기반으로 수정안을 제안하는 B2B SaaS 서비스입니다.

**노동톡**은 AI 기반 노동상담 서비스로, 근로자들의 노동법 관련 고민을 상담하고 필요시 전문 노무사를 연결해주는 수익 모델입니다.

### 핵심 가치 제안

| 기존 방식 | ContractPilot |
|----------|---------------|
| 변호사 검토 비용: 건당 200만원 | 월 9.9만원 구독 |
| 검토 시간: 2~3일 | 10분 내 분석 완료 |
| 법적 근거 확인: 별도 리서치 필요 | 한국 판례 자동 인용 |

## 주요 기능

### 1. 계약서 PDF 업로드 및 분석

사용자가 PDF 형식의 계약서를 드래그앤드롭으로 업로드하면, AI가 자동으로 텍스트를 추출하고 조항별로 분리하여 분석을 시작합니다.

**지원 형식**: PDF (최대 10MB)

**분석 대상 계약서 유형**:
- 투자계약서
- 근로계약서
- 용역계약서
- 임대차계약서
- 매매계약서
- 기타 상업 계약서

---

### 2. 조항별 위험도 분석

각 조항에 대해 1~10점 척도의 위험도 점수를 산출합니다.

| 위험 등급 | 점수 범위 | 의미 |
|----------|----------|------|
| Low (낮음) | 1~3점 | 일반적인 조항, 특별한 위험 없음 |
| Medium (보통) | 4~6점 | 주의가 필요한 조항, 검토 권장 |
| High (높음) | 7~10점 | 불리한 조항, 수정 강력 권장 |

**분석 항목**:
- 위험도 점수 및 등급
- 조항 요약
- 잠재적 문제점 목록
- 관련 한국 판례 인용
- 수정 제안 (고위험 조항)

---

### 3. 원본 vs 수정본 비교 뷰

고위험 조항에 대해 AI가 제안한 수정안을 원본과 나란히 비교할 수 있습니다.

**기능**:
- Side-by-Side 비교 화면
- 변경된 부분 하이라이트
- 조항별 개별 비교
- 원본/수정본 간 쉬운 전환

---

### 4. 수정된 계약서 다운로드 (Word)

분석 결과를 바탕으로 고위험 조항이 수정된 버전의 계약서를 Word 문서로 다운로드할 수 있습니다.

**특징**:
- 위험도 7점 이상 조항 자동 수정 적용
- 원본 형식 유지
- `.docx` 형식 지원
- 즉시 편집 가능

---

### 5. 분석 리포트 다운로드 (PDF)

전문적인 분석 리포트를 PDF 형식으로 생성하여 다운로드할 수 있습니다.

**리포트 구성**:

1. **표지**
   - 계약서 유형
   - 분석 일시
   - ContractPilot 브랜딩

2. **요약 (Executive Summary)**
   - 전체 위험도 등급
   - 총 조항 수 / 고위험 조항 수
   - 평균 위험도 점수

3. **조항별 상세 분석**
   - 원문 내용
   - 위험도 점수 및 등급
   - 발견된 문제점
   - 수정 제안 (해당 시)
   - 관련 판례

4. **면책 조항**
   - 법적 효력 고지
   - 전문가 상담 권고

---

### 6. 판례 기반 법률 상담 챗봇

계약서 분석 결과와 연동된 AI 챗봇으로 추가 법률 상담을 받을 수 있습니다.

**기능**:
- 계약서 분석 컨텍스트 반영
- 한국 판례 기반 답변
- 대화 히스토리 유지
- 자연어 질문 지원

**질문 예시**:
- "손해배상 조항이 불리한 것 같은데 어떻게 수정해야 하나요?"
- "이 계약서에서 가장 주의해야 할 조항은 무엇인가요?"
- "관련 판례가 더 있나요?"

---

### 7. 노동상담 AI 챗봇 (노동톡)

AI 기반 노동법 전문 상담 서비스로, 근로자의 다양한 노동 관련 고민을 상담합니다.

**상담 카테고리**:
- 임금체불 (월급, 수당, 퇴직금 미지급)
- 부당해고 (해고 통보, 권고사직 강요)
- 직장 내 괴롭힘 (폭언, 따돌림, 부당 업무지시)
- 산업재해 (업무상 부상, 질병)
- 근로조건 (연차, 휴가, 근로시간)
- 4대보험 (미가입, 체납)

**기능**:
- 공감 우선의 친근한 상담
- 근로기준법, 노동관계법령 기반 정확한 정보 제공
- 실용적이고 구체적인 해결 방법 안내
- 관련 노동 판례 인용
- 복잡한 케이스 시 전문가 연결 권유

**질문 예시**:
- "3개월째 월급을 못 받았어요"
- "갑자기 해고 통보를 받았어요"
- "상사가 계속 폭언을 해요"
- "산재 신청은 어떻게 하나요?"

---

### 8. 전문 노무사 연결 서비스

복잡한 노동 문제에 대해 전문 노무사와 상담을 연결해드립니다.

**연결이 권장되는 상황**:
- 금액이 큰 경우 (임금체불 300만원 이상)
- 법적 절차가 필요한 경우 (진정, 소송)
- 회사가 대응하지 않는 경우
- 복잡한 사안 (여러 문제가 얽힌 경우)
- 시급한 경우 (해고 예정, 시효 임박)

**신청 절차**:
1. 상담 중 "전문가 연결" 버튼 클릭
2. 이름, 연락처, 희망 상담 시간 입력
3. 개인정보 동의
4. 24시간 내 전문 노무사 연락

**수익 모델**: 노무사 연결 성공 시 수수료 발생

## 기술 스택

### Frontend
- **Next.js 14** - React 기반 풀스택 프레임워크
- **Tailwind CSS** - 유틸리티 퍼스트 CSS 프레임워크
- **Framer Motion** - 애니메이션 라이브러리
- **React Dropzone** - 파일 업로드 컴포넌트

### Backend
- **FastAPI** - 고성능 Python 웹 프레임워크
- **OpenAI GPT-4o** - 계약서 분석 및 챗봇 AI
- **PyPDF2** - PDF 텍스트 추출
- **python-docx** - Word 문서 생성
- **ReportLab** - PDF 리포트 생성

### 배포
- **Frontend**: Cloudflare Pages (https://contractpilot.pages.dev)
- **Backend**: Railway (https://contractpilot-production.up.railway.app)
